Projects

Project #1

TourEase

TourEase is a travel platform built with Node.js and Express.js for fast and smooth performance. It uses the Pug view engine for dynamic pages and secure user authentication with JWT. Data is managed using MongoDB with Mongoose, and Multer enables easy photo uploads. Various npm libraries are included to add features and improve the user experience.

HTML5, PUG, JavaScript, Node.js, Express, MongoDB

Project #2

MERN AUTH

MERN AUTH is a full-stack authentication system built with MongoDB, Express, React, and Node.js. It includes features like login, registration, forgot password, and email-based OTP for secure password resets, ensuring a smooth and secure user experience.

JavaScript, ReactJS, Tailwind CSS, Express, MongoDB
